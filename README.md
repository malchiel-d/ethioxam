# EUEE Simulator
 Web-based tool  mimicing the actual exam setup, so you can get a feel for the timing and the way questions are laid out before the real thing.

## features
- ​Real Exam Feel: A clean interface that focuses on the questions.
- Live Scoring: No waiting around, see how you did as soon as you finish.
- ​Timer: Helps you keep track of your pace so you don't get stuck on one question for too long.
- ​Mobile Friendly: Designed with a mobile-first approach, so practicing on a phone is as smooth as on a desktop.
