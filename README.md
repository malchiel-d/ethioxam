# 🎓 ETIOXAM: EUEE Exam Simulator

ETIOXAM is web-based tool designed to replicate the authentic Ethiopian University Entrance Examination (EUEE) experience. It provides students with a realistic preview of the digital exam environment, focusing on timing, question layout, and immediate performance feedback.
## 🚀 Live 
[malchiel-d.github.io/ethioxam](https://malchiel-d.github.io/ethioxam/)
 
## ✨ Core Features

* **Authentic Environment:** A distraction-free UI that mirrors the actual EUEE setup.
* **Instant Feedback:** Automated scoring engine provides results immediately upon submission.
* **Integrated Timer:** A real-time countdown to help students master their pacing.
* **Mobile-First Design:** Fully responsive layout that works across smartphones, tablets, and desktops.
* **Question Navigation:** A visual palette to quickly jump between questions and track progress.

## 🔧 Technical Stack

* **HTML5 & CSS3:** Semantic structure and modern layouts (Flexbox/Grid).
* **Vanilla JavaScript:** ES6+ logic for the scoring engine and timer.
* **Local Data:** Questions are managed via `data/data.js` for fast, offline-capable loading.
